Bundled Piper TTS voices (from piper-tts / OHF-Voice piper1-gpl).
Each voice is a pair: <id>.onnx and <id>.onnx.json

Shipped by default:
  en_US-lessac-medium  — US English, default agent voice
  en_US-ryan-low       — US English, alternate
  en_GB-cori-medium    — British English

To add more voices, see the project README (pip install piper-tts, python -m piper.download_voices …).
